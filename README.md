# SalesPulse - Sales Analytics Dashboard

SalesPulse is a sales analytics dashboard I built using Python, SQL, Streamlit and Plotly.

I created this project to practice working with a complete data analytics workflow: cleaning a dataset, loading it into a database, writing analysis logic, and showing the results in an interactive dashboard.

## Live Demo

https://sales-analytics-dashboard-yxslvgyrrkryclzyz4mnbgo.streamlit.app

## What the dashboard shows

The dashboard lets users explore sales data using filters for:

- Region
- Category
- Sub-Category

It includes:

- Total Sales, Total Profit, Sales Records and Profit Margin KPIs
- Sales by Region
- Sales by Category
- Profit vs Discount analysis
- Category and Sub-Category treemap
- Top 10 Sub-Categories by Sales
- Top 10 Sub-Categories by Profit
- Sales and Profit by Segment
- Sales and Profit by Ship Mode
- Filter-based business insights
- Download option for filtered data
